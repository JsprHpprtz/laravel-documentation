# Laravel Documentation Changelog

## [Initial Release] - 2026-01-03

### Added

- **Search Laravel Docs** - Browse 68+ documentation pages with favorites, recent pages, and version switching
- **Artisan Commands** - Reference 70+ Artisan commands organized by category
- **Laravel Snippets** - Copy 30+ code snippets for routes, models, migrations, and more
- **Create Laravel Project** - Wizard to create new Laravel projects with starter kit and database options
- **Run Artisan Command** - Execute common Artisan commands in your project directory

### Features

- Support for 17 Laravel versions (Master through 5.0)
- Favorites system for bookmarking documentation pages
- Recent pages tracking for quick access
- Laracasts integration with direct links to video tutorials
- Configurable Laravel project directory for running commands
- Cross-platform support (macOS and Windows)
