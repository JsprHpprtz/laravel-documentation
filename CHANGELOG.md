# Laravel Documentation Changelog

## [Initial Release] - 2026-01-03

- Added Search Laravel Docs command with 68 documentation pages across 10 categories
- Added support for 17 Laravel versions (Master through 5.0)
- Added favorites system for bookmarking documentation pages
- Added recent pages tracking for quick access
- Added Laracasts integration with direct links to video tutorials
- Added Artisan Commands reference with 56 commands across 9 categories
- Added Laravel Snippets with 32 code snippets for routes, models, migrations, and more
- Added Create Laravel Project wizard with starter kit and database selection
- Added Run Artisan Command for executing common commands directly from Raycast
- Added configurable Laravel version, browser behavior, and project directory preferences
